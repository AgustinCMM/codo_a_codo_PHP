Receta para preparar Merengue

Descripción general:

El merengue es un delicioso postre, un pastel pequeño o pastelito de forma ahovada o ahuevada que se hace batiendo las claras de huevo a punto de nieve, se lo mezcla con el almíbar, y se lo hornea veinte minutos. Una vez que está...

"Bueno, se puede danzar, mientras se hace el batimento, no hay problema."

Ingredientes:

  • Claras de huevo.
  • Azucar.
  • Agua.
  • Mermelada de fruta.
  • Crema Chantilly.
  • Fruta para decorar.
"No se lo puede partir, se desmigaja todo, es un postre seco."

Procedimiento:

  1. Se baten las claras de huevo a punto de nieve
  2. Se lo mezcla con el almíbar
  3. Y se lo hornea unos 20 minutos aproximadamente, hasta tener terminado ese pequeño pastel de forma ahovada llamado merengue, que lo vamos a dividir en dos mitades:
  4. La mitad inferior, o hemimerengue inferior, la podemos cubrir con un poco de mermelada de fruta, o crema Chantilly
  5. Y la mitad superior, o hemimerengue superior, o ut supra hemimerengue, la podemos decorar con una pequeña fruta, por ejemplo una sandía, en la parte de arriba...
  6. Una pequeña, de 8 kilos.
  7. Sí, a veces se aplasta el merengue, porque si usted pone...
